
# Joinchat - Enhanced click to chat for WhatsApp
Wordpress plugin

[![WP compatibility](https://plugintests.com/plugins/wporg/creame-whatsapp-me/wp-badge.svg)](https://plugintests.com/plugins/wporg/creame-whatsapp-me/latest)
[![PHP compatibility](https://plugintests.com/plugins/wporg/creame-whatsapp-me/php-badge.svg)](https://plugintests.com/plugins/wporg/creame-whatsapp-me/latest)

The most installed WordPress plugin for WhatsApp: a customizable floating button that turns your visitors into chats, leads, and sales.

## Features

- Floating WhatsApp button with app/web behavior and desktop QR support.
- Eye-catching CTAs with Rich Chat bubbles (links, buttons, images, random text).
- Dynamic start messages with variables such as `{TITLE}`, `{URL}`, `{PRODUCT}`, `{SKU}`.
- WooCommerce integration and product-level customization.
- Built-in analytics events for GA4, GTM, Facebook Pixel, and Google Ads.
- Lightweight frontend: vanilla JavaScript, deferred scripts, lazy-loaded styles.
- Per-page and archive-level settings for content targeting.

## Premium and AI

Joinchat is the base toolkit. Premium and AI extend it with:

- Country targeting rules.
- Multi-channel support (Telegram, Messenger, calls, and more).
- Multi-agent schedules and fallback CTAs.
- Chat Funnel flows.
- AI agents trained on your own content.

## Installation

1. Upload the entire `creame-whatsapp-me` folder to `/wp-content/plugins/`.
2. Activate the plugin from the WordPress Plugins screen.
